## How to run?
1. Clone Repository
```bash

```
2. Create virtual environment
```bash
python -m venv visaaa

```
3. Active virtual environment
```bash
.\visaaa\Scripts\Activate.ps1

```
4. Install all requirements
```bash
pip install -r requirements.txt
